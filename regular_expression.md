
# Regular Expression


| sign | meaning |
| ---  | --------|
| ^    | beginning |
| $    |  ending |
| [abd]| given "a" "b" "d"  |
| [^abd] | all except "a" "b" "d" |
| .  |  any 1 character     |
| [0-9]| numeric |
| *    |  0 ~ max |
| +    |  1 ~ max |
| ?    |  0 ,  1  |
| {m[,M]} | minimal, Maximal |
